# Employee Department UI

A responsive Angular 18 application to manage employees by department. This frontend is built as part of a full-stack assignment, connected to a Spring Boot backend.

## 🧩 Features

- 🔍 Search employees by department ID
- 📋 View all employees in a department
- 👤 View individual employee details
- ✏️ Update employee information
- 🗑️ Delete employee record
- 💡 Clean UI styled using Bootstrap 5 and UI Avatars

---

## 🚀 Technologies Used

- Angular 18 (standalone components)
- Bootstrap 5+
- RxJS
- TypeScript
- REST API integration with Spring Boot
